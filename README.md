Overview:

A simple Python command-line To-Do List application that helps users manage daily tasks efficiently. It supports task creation, editing, deletion, completion tracking, priorities, and due dates while storing data in a JSON file for persistence.

Features:

Add new tasks
Edit existing tasks
Delete tasks
Mark tasks as completed
Set task priorities (Low, Medium, High)
Add optional due dates
Persistent JSON storage
Simple menu-driven interface
Technologies Used:
Python 3
JSON
Datetime
OS Module

How to Run:

Make sure Python 3 is installed.
Run the program:
python "to-do list.py"
Use the menu to manage your tasks.

Project Structure:

to-do list.py – Main application
todos.json – Stores task data
